다음 찾기 ['F3']


[찾기] 대화 상자를 띄우지 않고, 바로 전에 했던 [찾기]를 아래 방향으로 다시 한 번 실행합니다.
찾을 내용은 이전에 [찾기]에서 지정했던 것이며, 현재 커서 위치에서 시작하여 [아래쪽]으로 찾습니다.

연습 하기

1. [편집-찾기]를 실행합니다.
2. [찾기] 대화 상자가 나타나면 [찾을 내용]에 현재 문서의 내용 중에 찾고자 하는 문자열을 입력합니다.
3. [찾기] 단추를 누르거나, 글자판의 를 누릅니다.
4. [찾을 내용]에 해당하는 낱말을 찾아 블록으로 설정하여 보여 줍니다. 만일 찾고자 하는 문자열이 없다면, 찾는 문자열이 없다고 알려 줍니다.
5. [닫기] 단추를 누르거나, 를 눌러 [찾기] 대화 상자를 닫습니다.
6. [편집-다음 찾기]를 실행하거나 ['F3']을 눌러 바로 전에 실행했던 것과 같은 조건으로 아래 방향으로 [찾기]를 다시 한 번 실행합니다.
