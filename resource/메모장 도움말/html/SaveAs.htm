다른 이름으로 저장 ['Ctrl' + 'Shift' + 'S']


현재 편집하고 있는 문서를 새로운 이름으로 바꾸어
디스크에 저장하고, 바꾼 이름으로 편집을 계속합니다.
[다른 이름으로 저장하기]를 실행하면 파일 이름뿐만 아니라 파일 형식도 바꾸어 저장할 수 있습니다.

연습 하기

1. 편집 중인 문서에 대하여 [파일-다른 이름으로 저장하기]를 실행합니다.
2. [다른 이름으로 저장하기] 대화 상자가 나타나면 [저장 위치]에서 파일을 저장할 드라이브나 폴더를 지정합니다.
3. 파일 목록 상자에서 파일을 저장할 폴더를 선택합니다.
4. [파일 이름] 입력 상자에 새로운 이름을 입력합니다.
5. [저장] 단추를 누르면 문서 탭 이름표와 제목 막대에 새로 저장한 이름이 표시됩니다.
